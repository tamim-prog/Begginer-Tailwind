Shrinking bundle sizes

Tailwind comes with so many classes. It doesn't make sence to send al lof those to our users browesers. With PostCSS and PurgeCSS, we can shrink the amount of CSS we sent to our users.

We'll learn how to only send the classes that we actually use so that our bundle sizes can drop to even below 10kb!

Using Talwind with React
